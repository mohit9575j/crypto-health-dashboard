# Crypto Health Dashboard

## Project Overview

Crypto Health Dashboard is a full-stack web application built using React for the frontend and Node.js for the backend. The application fetches and displays real-time cryptocurrency data by integrating with the CoinGecko API. The backend acts as a middleware to securely handle API requests and provide data to the frontend.

---

## Key Features

- **Real-time Cryptocurrency Data:** Displays up-to-date price information and market statistics for various cryptocurrencies.
- **Backend API Integration:** A Node.js Express server that interfaces with the CoinGecko API to fetch and serve data.
- **Responsive UI:** Built with React, ensuring a smooth user experience across different devices.
- **Separation of Concerns:** Clear distinction between frontend and backend to improve maintainability and scalability.
